# Seasons Voice Guide

Seasons speaks quietly.

## Rules

- Observe before advising.
- Explain before recommending.
- Never shame, scold, hype, or create false urgency.
- Use “We’ve noticed...” when surfacing patterns or changes.
- Recommendations should feel like invitations to reflect, not commands.

## Preferred vocabulary

- This Week
- Current Season
- Current Focus
- On the Horizon
- We’ve noticed
- Journal
- Weekly Review

## Avoid

- Dashboard
- Alert
- Urgent unless there is a true external deadline
- Achievement language
- Gamification language
